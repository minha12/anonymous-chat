üìù Thread Operations

POST /api/threads/{board}

Create a new thread on a specific board.

Request Body
{
  "text": "string",
  "delete_password": "string"
}
Response
{
  "_id": "string",
  "text": "string",
  "created_on": "date",
  "bumped_on": "date",
  "replies": []
}
Redirects to /b/{board} on success
GET /api/threads/{board}

Get the 10 most recently bumped threads with latest 3 replies.

Response
[{
  "_id": "string",
  "text": "string",
  "created_on": "date",
  "bumped_on": "date",
  "replies": [{
    "_id": "string",
    "text": "string",
    "created_on": "date"
  }]
}]
Excludes reported and delete_password fields
DELETE /api/threads/{board}

Delete a thread completely.

Request Body
{
  "thread_id": "string",
  "delete_password": "string"
}
Response

Text response: "success" or "incorrect password"

PUT /api/threads/{board}

Report a thread.

Request Body
{
  "thread_id": "string"
}
Response

Text response: "success"

üí¨ Reply Operations

POST /api/replies/{board}

Add a reply to a thread.

Request Body
{
  "thread_id": "string",
  "text": "string",
  "delete_password": "string"
}
Updates thread's bumped_on date
Redirects to /b/{board}/{thread_id}
GET /api/replies/{board}?thread_id={thread_id}

Get a thread with all replies.

Response
{
  "_id": "string",
  "text": "string",
  "created_on": "date",
  "bumped_on": "date",
  "replies": [{
    "_id": "string",
    "text": "string",
    "created_on": "date"
  }]
}
Excludes reported and delete_password fields
DELETE /api/replies/{board}

Delete a reply (changes text to '[deleted]').

Request Body
{
  "thread_id": "string",
  "reply_id": "string",
  "delete_password": "string"
}
Response

Text response: "success" or "incorrect password"

PUT /api/replies/{board}

Report a reply.

Request Body
{
  "thread_id": "string",
  "reply_id": "string"
}
Response

Text response: "success"
